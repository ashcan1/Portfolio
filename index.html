Taskapp

I built this app using CodeIgniter 4, modern and fast PHP MVC framework. This app features CRUD( create, read, update, delete) for database records. The app has user registration with account activation by email, login with remember me, user password reset by email and much more. By building this app I gained and improved my knowledge on OOP and CRUD, Separate presentation code, Authntication, SMTP, Ajax and JSON and deployment.

BEtheDJ

BE the Dj is an app that three team mates and I from Le Wagon built using Ruby on Rails as our final project. The app allows the bar owners to create their own music list from Spotify and allows customers to pick the music from their own device. The request then will go onto the Dj dashboard and the Dj will play the most requested tune! The app is a social jukebox for Spotify. It was a very challenging project the best bit was working within a team.

Meditation

This is my meditation app. I used Html, Css and JavaScript, I also worked with SVG files the app enables the user to select a timer out of the set options. I created this app to bring peace to the user and to demonstrate how easy it is to take time out and meditate to be able to open you mind and take time to apreciate your enviroment. This is a big topic in the current climate with mental health and wellbeing.
